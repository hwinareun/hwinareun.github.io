express
===
`npm i express` : express 다운로드.

http보다 간결한 코드로 서버 열 수 있음.

```jsx
const express = require('express')
const app = express()

 //GET 메서드로 '/test'이 날아오면 매개변수로
 // 매개변수로 전달받은 콜백 함수를 호출 => 서버에 셋팅.
 app.get('/test', function(req,res){
    res.send('TEST!')
})
 //서버 셋팅: 포트 넘버
app.listen(8080)
```
